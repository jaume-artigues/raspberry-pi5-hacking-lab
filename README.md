# 💻 Raspberry Pi 5 - Hacking Lab (Laboratorio de Hacking)

Configuración inicial de un laboratorio de pruebas en una Raspberry Pi 5 con arranque desde NVMe.

---

## 🇪🇸 Versión en Español

### 🛠️ Especificaciones del Hardware
| **Modelo** | Raspberry Pi 5 (16GB RAM) |
| **Procesador** | Broadcom BCM2712 Quad-core ARM Cortex-A76 @ 2.4GHz |
| **Almacenamiento** | 500GB Crucial P3 Plus NVMe SSD |
| **Interfaz** | PCIe Gen 3 mediante NVMe HAT |
| **Refrigeración** | Ventilador oficial (Active Cooler) y caja original |
| **S.O.** | Debian 13 (Trixie) 64-bit |

### 🎯 Objetivo del Proyecto
Crear un entorno seguro y eficiente para prácticas de ciberseguridad y hacking ético. Consolidar y poner en práctica mis conocimientos de redes, sistemas y ciberseguridad.

### 📝 Writeup
- Configuración inicial de Debian 13.
- Montaje y optimización del NVMe.
- Herramientas instaladas:
    - UFW (Uncomplicated Firewall)
